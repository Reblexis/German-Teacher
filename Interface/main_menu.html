German Teacher is an open-source project that's designed to help students with learning german rules and vocabulary.
There is also an access to analytics regarding your performance and improvement.

Please keep in mind that this project is still under development and there may be some bugs or incomplete features.
If you find any bugs or if you are interested in contributing to the project, please contact me at my email: viktor.cihal@gmail.com.
